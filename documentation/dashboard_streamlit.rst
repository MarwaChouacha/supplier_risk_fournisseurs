Dashboard Streamlit
===================

Fonctionnalités
---------------

Le tableau de bord permet :

- Analyse des KPIs fournisseurs
- Visualisation des risques
- Classement des fournisseurs
- Monitoring des modèles MLflow

Cas d’Usage
-----------

- Suivi des fournisseurs critiques
- Simulation de scénarios
- Aide à la prise de décision stratégique
