Génération de Données Synthétiques
=================================

Objectif
--------

L’objectif de cette phase est de créer des jeux de données réalistes simulant le
comportement des fournisseurs dans un contexte industriel.

Méthodologie
------------

Les données sont générées en intégrant :

- Retards de livraison
- Défauts qualité
- Quantités livrées
- Indicateurs financiers
- Certifications et conformité
- Risques géopolitiques

Catégories de Variables
----------------------

- Informations générales fournisseur
- Indicateurs financiers
- Indicateurs de livraison (OTD)
- Indicateurs qualité
- Conformité et certifications
- Risques supply chain

Versioning des Données
----------------------

Les datasets sont versionnés à l’aide de **DVC**, assurant :

- Traçabilité complète
- Reproductibilité
- Gestion des évolutions de données
