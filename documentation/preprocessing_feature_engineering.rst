Preprocessing et Feature Engineering
====================================

Nettoyage des Données
--------------------

- Gestion des valeurs manquantes
- Normalisation des variables numériques
- Traitement des valeurs aberrantes
- Catégorisation des retards

Feature Engineering
-------------------

Plusieurs types de features ont été créés :

- **Features temporelles** (historique fournisseur)
- **Features composites** (scores agrégés)
- **Features d’interaction** (qualité × délai)
- **Features binaires** (conformité, certification)
- **Encodage catégoriel** (One-Hot, Label Encoding)

Ces transformations améliorent la capacité prédictive des modèles.
