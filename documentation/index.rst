Supplier Risk Detection Documentation
=====================================

Bienvenue dans la documentation officielle du projet **Supplier Risk Prediction System**.

Ce projet vise à anticiper les risques fournisseurs à l’aide de techniques avancées
de **Data Science**, **Machine Learning** et **MLOps**, afin d’améliorer la prise de décision
dans les chaînes d’approvisionnement industrielles.
